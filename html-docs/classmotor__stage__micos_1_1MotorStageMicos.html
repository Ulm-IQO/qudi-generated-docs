◆ get_constraints()

def motor_stage_micos.MotorStageMicos.get_constraints (   self)

Retrieve the hardware constrains from the motor device.

Returns
dict: dict with constraints for the sequence generation and GUI

Provides all the constraints for the motorized stage (like total movement, velocity, ...) Each constraint is a tuple of the form (min_value, max_value, stepsize)

The possible keys in the constraint are defined here in the interface file. If the hardware does not support the values for the constraints, then insert just None. If you are not sure about the meaning, look in other hardware files to get an impression.
